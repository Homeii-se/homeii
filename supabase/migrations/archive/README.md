# Arkiverade V1-migrationer

Detta är historiska migration-filer från V1 av Mina sidor
(mätarpunkts-baserad datamodell). De är inte längre aktiva.

V2-schemat (hem-baserad datamodell) lever i
`supabase/schema.sql` och körs som ett komplett drop-och-create-script
i Supabase SQL Editor. Migrations-mappen används inte längre för
sequential migrations.

Filerna här bevaras för historisk referens. För att förstå nuvarande
schema, läs `supabase/schema.sql`.
